## üîß Running the Projects

To run a project, follow these steps:

1. **Install Node.js**: Visit the [Node.js website](https://nodejs.org/) to download and install the latest version.

2. **Open Terminal or Command Prompt**:
   - On Windows, press `Win + R`, type `cmd`, and hit Enter.
   - On macOS, locate Terminal in your Applications folder.
   - On Linux, open your preferred terminal emulator.

3. **Navigate to the project folder**:
   Use the `cd` command to change directories. For example:
   ```
   cd path/to/your/extracted/project-folder
   ```

4. **Install Dependencies**:
   Run this command in your terminal:
   ```
   npm install
   ```
   This command fetches all necessary packages for the project.

5. **Start the Development Server**:
   After the dependencies install, type this command:
   ```
   npm start
   ```
   Your application will open in your web browser automatically. If it doesn‚Äôt, open your browser and enter `http://localhost:3000`.
